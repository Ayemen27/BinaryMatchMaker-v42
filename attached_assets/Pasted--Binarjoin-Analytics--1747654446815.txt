وصفحاته المختلفة:

منصة بينارجوين للتحليلات (Binarjoin Analytics)
هذا التطبيق هو منصة تداول ذكية متخصصة للأسواق العربية، تقدم تحليلات مالية متقدمة وإشارات تداول دقيقة باستخدام تقنيات التحليل الفني والذكاء الاصطناعي.

الميزات الرئيسية للتطبيق
التحليل الفني المتقدم: يستخدم مؤشرات فنية متعددة مثل RSI وMACD والمتوسطات المتحركة لتوليد إشارات تداول دقيقة
نظام استعادة تلقائي: يحتوي على آلية استعادة سريعة لضمان استمرارية عمل النظام
نسخ احتياطي دوري: يحفظ آخر 10 نسخ من البيانات كل 10 دقائق
واجهة مستخدم تفاعلية: تقدم تجربة مستخدم سلسة ومتجاوبة
دعم API الخارجية: تكامل مع Binance API للحصول على بيانات السوق المباشرة
دعم متعدد اللغات: متوفر باللغتين العربية والإنجليزية
الصفحات الرئيسية في التطبيق
1. صفحات المصادقة والتسجيل
صفحة تسجيل الدخول: تتيح للمستخدمين إدخال بريدهم الإلكتروني وكلمة المرور للدخول
صفحة التسجيل: لإنشاء حساب جديد مع إدخال البيانات الشخصية واختيار خطة الاشتراك
صفحة استعادة كلمة المرور: لاستعادة كلمة المرور المنسية
2. لوحة التحكم (الصفحة الرئيسية)
تظهر بعد تسجيل الدخول وتعرض:

ملخص السوق: أحدث أسعار الأصول المالية الرئيسية
إشارات اليوم: إشارات التداول الحالية والمقترحة
بطاقات إحصائيات سريعة: تعرض معلومات مثل الإشارات النشطة وإجمالي الأرباح ودقة الإشارات وحالة الاشتراك
بطاقات الميزات الرئيسية: للوصول السريع إلى الميزات المهمة مثل تحليل السوق وإدارة الإشارات والإعدادات
جدول آخر الإشارات: يعرض أحدث إشارات التداول مع تفاصيلها
3. صفحة الإشارات
عرض قائمة الإشارات: مع إمكانية تصفيتها حسب نوع الأصل واتجاه الإشارة والمدة الزمنية
تفاصيل الإشارة: تعرض معلومات شاملة عن كل إشارة مثل:
معلومات الأصل
اتجاه التداول الموصى به
نقاط الدخول والخروج
وقف الخسارة
التحليل الفني المدعم للإشارة
الرسوم البيانية ذات الصلة
متابعة الإشارات: إمكانية متابعة إشارات محددة وتلقي تنبيهات عند الدخول أو الخروج
4. صفحة تحليلات السوق
الرسوم البيانية: لتحليل الأصول المالية مع اختيار الإطار الزمني
المؤشرات الفنية: مجموعة من المؤشرات الفنية المتاحة (RSI، MA، MACD، البولنجر باند، الستوكاستيك)
تقارير السوق: تقارير يومية وأسبوعية وتحليلات الاتجاهات وتوقعات السوق
5. صفحة الاشتراك
عرض خطط الاشتراك: الخطة الأساسية والاحترافية وVIP
إدارة الاشتراك الحالي: عرض تفاصيل الاشتراك الحالي، ترقية أو تجديد الاشتراك
سجل المدفوعات: عرض سجل المدفوعات السابقة
6. صفحة الإعدادات والملف الشخصي
تحديث المعلومات الشخصية
تفضيلات الإشعارات: اختيار أنواع الإشعارات وطريقة تلقيها
تغيير كلمة المرور
تفضيلات اللغة
تفعيل المصادقة الثنائية
مراجعة نشاط الحساب
7. صفحة الدعم والمساعدة
مركز المساعدة: يتضمن الأسئلة الشائعة والدروس التعليمية وأدلة استخدام المنصة
نموذج الاتصال بالدعم
صفحة تقديم الاقتراحات
التقنيات المستخدمة في المشروع
الواجهة الأمامية: React.js مع TypeScript و Tailwind CSS
الخادم الخلفي: Node.js مع Express
قاعدة البيانات: PostgreSQL مع Drizzle ORM
المصادقة: نظام مصادقة متكامل مع دعم للمصادقة الثنائية
التعريب: دعم متعدد اللغات باستخدام i18next
الاتصال الخارجي: تكامل مع Binance API وOpenAI API (للتحليل المتقدم)
هذا التطبيق يقدم حلاً متكاملاً للمتداولين في الأسواق العربية، مع التركيز على تقديم تحليلات دقيقة وإشارات تداول موثوقة، مدعومة بواجهة مستخدم سهلة الاستخدام ونظام قوي للدعم والمساعدة.
